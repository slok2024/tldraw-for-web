# tldraw-for-web

1. 下载 Release 里的 dist.zip
2. 解压到 Web 服务器目录
3. 确保通过 HTTP 协议访问 (如 Nginx 或 http-server)

tldraw 是目前前端开发圈最火的开源白板项目之一。不同于传统的画图软件（如 Windows 画图），它是一个基于向量的、无限缩放的协作空间。

功能： 支持手绘、几何图形、箭头连接线、文字标注、图片/视频嵌入。

技术栈： 基于 React 和 TypeScript 构建，拥有极高的性能（即使画布上有数千个图形也不会卡顿）。
